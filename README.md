
# Cloud Psychiatric Care Landing Page

Note: This project was developed for a real healthcare practice and is actively hosted through Netlify. The website itself does not store patient information and primarily serves as an informational landing page with links to external patient systems.

A professional healthcare practice website designed to provide patients with easy access to provider information, practice details, available services, and appointment resources.

## Overview

This project was developed as a custom website for a Cloud Psychiatric Care. The site serves primarily as a landing page and informational resource for current and prospective patients.

Key features include:

- Practice information and provider biographies
- Services and treatment information
- Links to external patient portals and resources
- Integrated patient engagement chatbot
- Responsive design for desktop and mobile devices
- Simple navigation focused on patient accessibility

## Hosting

This website is hosted using Netlify's free hosting tier.

Because the site functions as an informational landing page and does not store patient records, medical information, or customer data, a lightweight static hosting solution was selected. External patient portals and third-party systems are used for any patient-specific interactions.
